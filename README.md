## 🛠️ The Arsenal

### 🕵️‍♂️ 1. Dynamic Stealth Scraper (`dynamic_stealth_scraper.py`)
* **Technology:** `Python` | `Playwright`
* **Purpose:** Bypasses heavy anti-bot security by utilizing a physical headless Chromium browser to navigate dynamic, JavaScript-heavy domains. Extracts DOM elements exactly as they render for human users.

### 🤖 2. Robotic Authentication Bot (`automated_login_bot.py`)
* **Technology:** `Python` | `Playwright`
* **Purpose:** An RPA engine designed to physically interact with web portals, inject authorized credentials, and bypass login gateways to facilitate secure behind-the-wall daily reporting or data entry.

### 🔌 3. JSON API Data Miner (`json_api_extractor.py`)
* **Technology:** `Python` | `Requests`
* **Purpose:** High-speed backend integration. Bypasses frontend HTML entirely to interface directly with server APIs, extracting clean dictionary data for rapid dataset compilation.

### ⚡ 4. Directory Lead Extractor (`directory_lead_extractor.py`)
* **Technology:** `Python` | `BeautifulSoup4` | `Requests`
* **Purpose:** A lightweight, high-speed static HTML parser. Designed to rapidly scan local business directories and e-commerce listings to generate structured CSV lead sheets and price-tracking reports.
